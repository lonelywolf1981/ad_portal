# Модуль crypto.py

## Назначение
Модуль для криптографических операций, таких как шифрование и дешифрование данных.

## Функциональность
- Шифрование и дешифрование строк с использованием Fernet
- Генерация ключей для шифрования
- Обработка ошибок шифрования и дешифрования

## Взаимодействие с другими модулями
- Используется в сервисах приложения для безопасного хранения чувствительных данных
- Может использоваться в модуле настроек для шифрования паролей
