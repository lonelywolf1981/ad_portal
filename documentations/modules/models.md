# Модуль models.py

## Назначение
Модуль, определяющий ORM-модели для базы данных с использованием SQLAlchemy.

## Функциональность
- Определение моделей для таблиц базы данных:
  - `LoginAudit` - журнал аудита входа
  - `LocalUser` - локальные пользователи
  - `AppSettings` - настройки приложения
  - `UserPresence` - информация о присутствии пользователей
  - `HostUserMap` - сопоставление пользователей и хостов
- Определение полей и отношений между моделями

## Взаимодействие с другими модулями
- Использует базовый класс из `db` для наследования моделей
- Используется модулем `repo` для выполнения запросов к базе данных
- Используется модулем `schema` для создания таблиц
- Используется различными компонентами приложения для работы с данными
