# Модуль timezone_utils.py

## Назначение
Модуль для работы с часовыми поясами и временными зонами.

## Функциональность
- Вспомогательные функции для преобразования временных меток
- Обработка временных зон
- Конвертация времени между различными часовыми поясами

## Взаимодействие с другими модулями
- Используется в различных компонентах приложения для корректной обработки временных данных
- Может использоваться в логировании, аудите и других модулях, работающих с временными метками
