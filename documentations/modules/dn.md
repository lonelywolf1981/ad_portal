# Модуль dn.py

## Назначение
Модуль для работы с Distinguished Names (DN) в Active Directory.

## Функциональность
- Преобразование DN в URL-безопасный идентификатор
- Преобразование URL-безопасного идентификатора обратно в DN
- Извлечение значения первого компонента DN (например, из CN=USB-Deny,OU=... извлекает USB-Deny)

## Взаимодействие с другими модулями
- Используется модулем `ad_management` для работы с идентификаторами пользователей и групп
- Используется в различных частях приложения для безопасной передачи DN в URL и HTML
