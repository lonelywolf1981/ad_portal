# Модуль net_scan.py

## Назначение
Модуль для сканирования сети и обнаружения присутствия пользователей на хостах.

## Функциональность
- Сканирование сети по заданным CIDR
- Обнаружение пользователей на сетевых хостах
- Использование различных методов (WinRM, WMI, SMB) для определения присутствия
- Обновление информации о присутствии пользователей

## Взаимодействие с другими модулями
- Используется в фоновых задачах для периодического сканирования
- Взаимодействует с базой данных через модели для сохранения результатов
- Может использовать настройки из AppSettings для конфигурации сканирования
