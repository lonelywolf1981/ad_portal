# Модуль ad_utils.py

## Назначение
Модуль вспомогательных функций для работы с Active Directory.

## Функциональность
- Вспомогательные функции для обработки данных AD
- Функции для преобразования и форматирования данных
- Утилиты для работы с атрибутами пользователей и групп

## Взаимодействие с другими модулями
- Используется модулем `ad.client` для вспомогательных операций
- Может использоваться в маршрутах для обработки данных AD
