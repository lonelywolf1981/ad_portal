# Модуль presence.py

## Назначение
Модуль для маршрутов, связанных с информацией о присутствии пользователей.

## Функциональность
- Отображение информации о присутствии пользователей
- Получение данных о том, где пользователи в настоящее время находятся
- Обновление информации о присутствии
- Предоставление API для получения информации о присутствии

## Взаимодействие с другими модулями
- Использует зависимости из `deps` для проверки авторизации
- Взаимодействует с базой данных через модели UserPresence и HostUserMap
- Может использовать данные из net_scan для обновления информации о присутствии
