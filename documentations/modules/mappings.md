# Модуль mappings.py

## Назначение
Модуль для определения сопоставлений, используемых в приложении.

## Функциональность
- Определение сопоставлений между различными типами данных
- Определение констант и словарей сопоставления
- Централизованное хранение сопоставлений для использования в разных частях приложения

## Взаимодействие с другими модулями
- Используется различными компонентами приложения для преобразования значений
- Может использоваться в маршрутах, моделях и службах
