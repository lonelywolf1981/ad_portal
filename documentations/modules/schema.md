# Модуль schema.py

## Назначение
Модуль для управления схемой базы данных и миграциями.

## Функциональность
- Обеспечение актуальности схемы базы данных
- Создание отсутствующих таблиц
- Добавление новых столбцов в таблицу настроек
- Поддержка минимальных миграций для SQLite

## Взаимодействие с другими модулями
- Использует `engine` и `Base` из `db` для работы с базой данных
- Используется модулем `bootstrap` для инициализации схемы
- Используется как в веб-приложении, так и в Celery-воркере
