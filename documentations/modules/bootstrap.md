# Модуль bootstrap.py

## Назначение
Модуль инициализации приложения, содержащий логику начальной настройки, которая ранее находилась в main.py.

## Функциональность
- Инициализация приложения с необходимыми шагами настройки
- Обеспечение актуальности схемы базы данных
- Создание администратора при первом запуске приложения

## Взаимодействие с другими модулями
- Использует `get_env` из `env_settings` для получения конфигурационных переменных
- Использует `db_session` и `ensure_bootstrap_admin` из `repo` для работы с базой данных
- Использует `ensure_schema` из `schema` для обеспечения актуальности схемы БД
- Использует `hash_password` из `security` для хеширования пароля администратора
