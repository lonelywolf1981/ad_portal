# Модуль session.py

## Назначение
Модуль для управления сессиями пользователей с использованием подписанных токенов.

## Функциональность
- Создание сессионных токенов с использованием URLSafeTimedSerializer
- Чтение и проверка срока действия сессионных токенов
- Обработка различных форматов данных сессии (обернутые и плоские)

## Взаимодействие с другими модулями
- Использует `get_env` из `env_settings` для получения секретного ключа
- Используется модулем `deps` для проверки сессии пользователя
- Используется в аутентификационных процессах приложения
