# Модуль db.py

## Назначение
Модуль для настройки подключения к базе данных SQLite с использованием SQLAlchemy.

## Функциональность
- Создание движка базы данных
- Настройка сессии базы данных
- Установка pragma-параметров SQLite (WAL режим, таймаут)
- Создание базового класса для моделей

## Взаимодействие с другими модулями
- Использует `get_env` из `env_settings` для получения пути к базе данных
- Используется модулем `models` как базовый класс для моделей
- Используется модулем `repo` для создания сессий базы данных
- Используется модулем `schema` для работы с метаданными
