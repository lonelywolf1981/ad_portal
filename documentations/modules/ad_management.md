# Модуль ad_management.py

## Назначение
Модуль маршрутов для управления Active Directory пользователями и группами.

## Функциональность
- Поиск и отображение пользователей AD
- Поиск и отображение групп AD
- Отображение подробной информации о пользователе
- Обновление атрибутов пользователя
- Разблокировка учетной записи пользователя
- Включение/отключение учетной записи пользователя
- Удаление пользователя из AD
- Добавление пользователя в группу AD
- Удаление пользователя из группы AD
- Отображение участников группы
- Создание новых пользователей и групп
- Управление участниками групп

## Взаимодействие с другими модулями
- Использует `require_session_or_hx_redirect` и `require_initialized_or_redirect` из `deps` для проверки авторизации
- Использует `ADClient` из `ad` для взаимодействия с AD
- Использует `db_session` и `get_or_create_settings` из `repo` для работы с настройками
- Использует `ad_mgmt_cfg_from_settings` из `services` для получения конфигурации AD
- Использует `audit_login` из `services.audit` для аудита операций
- Использует `dn_to_id`, `id_to_dn`, `dn_first_component_value` из `utils.dn` для работы с DN
- Использует `htmx_alert`, `templates`, `ui_result` из `webui` для возврата результатов
