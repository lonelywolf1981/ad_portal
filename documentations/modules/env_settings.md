# Модуль env_settings.py

## Назначение
Модуль для управления настройками окружения приложения с использованием Pydantic.

## Функциональность
- Определение класса `EnvSettings` с настройками приложения
- Использование LRU-кеширования для эффективного получения настроек
- Определение различных параметров: секретный ключ, настройки базы данных, учетные данные администратора и т.д.

## Взаимодействие с другими модулями
- Используется модулем `bootstrap` для получения настроек окружения
- Используется модулем `db` для получения пути к базе данных
- Используется различными компонентами приложения для получения конфигурационных параметров
