# Модуль webui.py

## Назначение
Модуль для веб-интерфейса пользователя, включающий шаблоны и вспомогательные функции для UI.

## Функциональность
- Определение шаблонов Jinja2
- Вспомогательные функции для генерации HTML-ответов
- Функции для работы с HTMX
- Вспомогательные функции для отображения результатов операций

## Взаимодействие с другими модулями
- Используется маршрутизаторами для возврата HTML-ответов
- Использует шаблоны из папки templates
- Взаимодействует с различными компонентами приложения для отображения данных
