# Модуль celery_app.py

## Назначение
Модуль для настройки и инициализации приложения Celery.

## Функциональность
- Настройка конфигурации Celery
- Определение брокера сообщений
- Настройка параметров выполнения задач
- Инициализация экземпляра приложения Celery

## Взаимодействие с другими модулями
- Используется для выполнения фоновых задач
- Взаимодействует с модулем tasks для регистрации задач
- Может использовать настройки из env_settings
