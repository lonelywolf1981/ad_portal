# Модуль main.py

## Назначение
Основной модуль приложения FastAPI, который инициализирует веб-приложение, настраивает маршруты, промежуточное ПО и жизненный цикл приложения.

## Функциональность
- Создание экземпляра FastAPI с названием "AD Portal"
- Настройка CSRF-защиты для POST/PUT/DELETE/PATCH запросов
- Подключение статических файлов
- Регистрация маршрутов для различных компонентов приложения
- Настройка жизненного цикла приложения (lifespan) для инициализации при старте

## Взаимодействие с другими модулями
- Импортирует `initialize_application` из `bootstrap` для инициализации при старте
- Подключает различные маршруты из папки `routers`:
  - `health` - проверка работоспособности
  - `auth` - аутентификация
  - `index` - главная страница
  - `settings` - настройки
  - `users` - пользователи
  - `hosts` - хосты
  - `presence` - присутствие
  - `ad_management` - управление AD
- Использует middleware для защиты от CSRF-атак
- Монтирует статические файлы из папки `app/static`
