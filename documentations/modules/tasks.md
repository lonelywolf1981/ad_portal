# Модуль tasks.py

## Назначение
Модуль для определения фоновых задач, выполняемых с помощью Celery.

## Функциональность
- Определение асинхронных задач
- Обработка периодических операций
- Задачи для сканирования сети и обновления информации о присутствии
- Обработка задач по обслуживанию системы

## Взаимодействие с другими модулями
- Использует компоненты приложения для выполнения фоновых операций
- Взаимодействует с базой данных через модули repo и models
- Может использовать службы из папки services
