# Модуль deps.py

## Назначение
Модуль зависимостей FastAPI, содержащий зависимости для маршрутов.

## Функциональность
- Получение конфигурации AD из настроек
- Проверка текущего пользователя по сессии
- Требование сессии или перенаправление на страницу входа
- Требование прав доступа к настройкам
- Требование инициализации приложения

## Взаимодействие с другими модулями
- Использует `read_session` из `session` для проверки сессии
- Использует `db_session` и `get_or_create_settings` из `repo` для получения настроек
- Использует `ad_cfg_from_settings` из `services` для получения конфигурации AD
- Используется маршрутизаторами для проверки авторизации
