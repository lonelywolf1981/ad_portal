# Модуль security.py

## Назначение
Модуль для обеспечения безопасности приложения, включая хеширование паролей.

## Функциональность
- Хеширование паролей с использованием алгоритма argon2
- Проверка соответствия пароля хешу
- Настройка контекста криптографии

## Взаимодействие с другими модулями
- Используется модулем `bootstrap` для хеширования пароля администратора
- Используется модулем `repo` при создании/проверке пользователей
- Используется в аутентификационных процессах приложения
