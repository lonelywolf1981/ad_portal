# Модуль client.py

## Назначение
Клиент для взаимодействия с Active Directory через LDAP.

## Функциональность
- Подключение к серверу AD
- Аутентификация и проверка подлинности
- Поиск пользователей и групп
- Получение подробной информации о пользователе
- Управление атрибутами пользователя
- Создание и удаление пользователей
- Управление принадлежностью к группам
- Блокировка и разблокировка учетных записей
- Создание и удаление групп
- Тестирование подключения к AD

## Взаимодействие с другими модулями
- Использует `Server`, `Connection`, `Tls` и другие из библиотеки ldap3
- Использует исключения из `ldap3.core.exceptions`
- Использует `ADConfig`, `ADUser` из `models`
- Использует `escape_ldap_filter_value`, `filetime_to_dt_str` из `utils`
